1. Create a new website containing 4 pages/sections

  - 4 sections( 4 containers with ids in the same page)
  
2. 4 pages/sections:
        - home page
        - about page
        - projects page
        - contact page
        
 3. Make the page responsive with at least 2 screen sizes width ( 0 - 800px) (800px - any dimension)
